## 为什么学 STL

STL 是 C++ 的标准模板库，提供了大量现成的数据结构和算法。在竞赛中熟练使用 STL 可以大幅减少编码时间，把精力集中在算法设计上。

## 容器

### vector — 动态数组

最常用的容器，没有之一。

```cpp
#include <vector>
using namespace std;

vector<int> v = {3, 1, 4, 1, 5};

// 常用操作
v.push_back(9);          // 尾部添加
v.pop_back();            // 尾部删除
v.size();                // 大小
v.empty();               // 是否为空
v[0];                    // 随机访问
v.front();               // 首元素
v.back();                // 尾元素

// 排序
sort(v.begin(), v.end());

// 去重（先排序）
v.erase(unique(v.begin(), v.end()), v.end());

// 二维向量
vector<vector<int>> grid(n, vector<int>(m, 0));
```

### set / multiset — 有序集合

```cpp
#include <set>
set<int> s;           // 去重
multiset<int> ms;     // 可重复

s.insert(3);
s.insert(1);
s.insert(4);
s.count(3);           // 1（存在）或 0（不存在）
s.find(3) != s.end(); // 是否存在
s.erase(3);           // 删除元素

// 最小值 / 最大值
*s.begin();           // 最小值
*s.rbegin();          // 最大值

// multiset 删除所有等于 x 的元素
ms.erase(ms.lower_bound(x), ms.upper_bound(x));
// 只删除一个
ms.erase(ms.find(x));
```

### map / unordered_map — 映射

```cpp
#include <map>
#include <unordered_map>

map<string, int> mp;              // 有序，O(log n)
unordered_map<string, int> ump;   // 无序，O(1) 平均

mp["hello"] = 1;
mp.count("hello");    // 是否存在
mp.erase("hello");    // 删除

// 遍历
for (auto& [key, val] : mp) {
    cout << key << ": " << val << endl;
}
```

### priority_queue — 优先队列

```cpp
#include <queue>

// 大顶堆（默认）
priority_queue<int> pq;

// 小顶堆
priority_queue<int, vector<int>, greater<int>> min_pq;

pq.push(3);
pq.push(1);
pq.push(4);
pq.top();   // 4（大顶堆）
pq.pop();

// 自定义排序（结构体）
struct Node {
    int val, idx;
    bool operator<(const Node& o) const {
        return val > o.val;  // 小顶堆
    }
};
priority_queue<Node> pq2;
```

### deque — 双端队列

```cpp
#include <deque>
deque<int> dq;

dq.push_front(1);  // 头部插入
dq.push_back(2);   // 尾部插入
dq.pop_front();    // 头部删除
dq.pop_back();     // 尾部删除
dq[0];             // 随机访问
```

### stack / queue

```cpp
#include <stack>
#include <queue>

stack<int> stk;
stk.push(1);
stk.top();
stk.pop();

queue<int> q;
q.push(1);
q.front();
q.pop();
```

## 算法

### sort — 排序

```cpp
#include <algorithm>

vector<int> v = {3, 1, 4, 1, 5};

// 默认升序
sort(v.begin(), v.end());

// 降序
sort(v.begin(), v.end(), greater<int>());

// 自定义比较
sort(v.begin(), v.end(), [](int a, int b) {
    return a > b;
});

// 结构体排序
struct Student {
    string name;
    int score;
};
vector<Student> students;
sort(students.begin(), students.end(), [](const Student& a, const Student& b) {
    if (a.score != b.score) return a.score > b.score;
    return a.name < b.name;
});
```

### lower_bound / upper_bound — 二分查找

```cpp
vector<int> v = {1, 2, 2, 3, 4};

// lower_bound: 第一个 >= x 的位置
auto it = lower_bound(v.begin(), v.end(), 2); // 指向 v[1]

// upper_bound: 第一个 > x 的位置
auto it2 = upper_bound(v.begin(), v.end(), 2); // 指向 v[3]

// 计算 2 的个数
int cnt = upper_bound(v.begin(), v.end(), 2) - lower_bound(v.begin(), v.end(), 2);

// 在降序数组中查找
vector<int> desc = {5, 4, 3, 2, 1};
auto it3 = lower_bound(desc.begin(), desc.end(), 3, greater<int>());
```

### next_permutation — 全排列

```cpp
vector<int> v = {1, 2, 3};
do {
    for (int x : v) cout << x << " ";
    cout << endl;
} while (next_permutation(v.begin(), v.end()));
```

### accumulate — 求和

```cpp
#include <numeric>

vector<int> v = {1, 2, 3, 4, 5};
int sum = accumulate(v.begin(), v.end(), 0);  // 15

// 注意初始值类型决定返回类型
long long sum2 = accumulate(v.begin(), v.end(), 0LL); // 用 long long
```

## 实用技巧

### 技巧 1：用 vector 代替固定数组

```cpp
// 不好
int a[100005];

// 好
vector<int> a(100005);
```

### 技巧 2：用 pair 做复合排序

```cpp
vector<pair<int, int>> v = {{3, 1}, {1, 3}, {1, 2}};
sort(v.begin(), v.end());
// 先按 first 升序，再按 second 升序
// 结果：(1,2), (1,3), (3,1)
```

### 技巧 3：用 set 模拟有序多重集合

```cpp
// 用 pair<值, 唯一id> 来区分相同值
set<pair<int, int>> s;
int id = 0;
s.insert({3, id++});
s.insert({3, id++});  // 两个 3 都能插入
```

### 技巧 4：用 bitset 优化状态压缩

```cpp
#include <bitset>
bitset<1005> dp;
dp[0] = 1;
// 滚动更新
dp = dp | (dp << w);
```

### 技巧 5：快速读入

```cpp
inline int read() {
    int x = 0, f = 1;
    char ch = getchar();
    while (ch < '0' || ch > '9') {
        if (ch == '-') f = -1;
        ch = getchar();
    }
    while (ch >= '0' && ch <= '9') {
        x = x * 10 + ch - '0';
        ch = getchar();
    }
    return x * f;
}
```

## 总结

| 场景 | 推荐容器 |
|------|----------|
| 需要随机访问 | `vector` |
| 需要有序 + 去重 | `set` |
| 需要有序 + 可重复 | `multiset` |
| 需要键值映射 | `unordered_map`（无序）或 `map`（有序） |
| 需要快速取最值 | `priority_queue` |
| 需要双端操作 | `deque` |

熟练掌握这些 STL 工具，能让你在竞赛中事半功倍。多用、多练，自然就熟了。
